
# Api Django - Municipality work


## API Reference
#### Get all proveedores

```http
  GET /api/proveedor
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |

#### Get a one proveedor

```http
  GET /api/proveedor/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
